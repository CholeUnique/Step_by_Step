## 本次目标
ES6 支持（Babel 转译）+ 数据结构可视化升级（diff-based stack/queue + React Flow 渲染 tree/linkedlist）

## 完成内容
- 安装依赖：`@babel/standalone`、`reactflow`
- 新增 `src/core/codeTransformer.js`：调用 Babel.transform 将 ES6+ 转为 ES5，失败时回退原始代码
- 修改 `InterpreterController.js`：在 `init()` 中调用 `transformCode()` 后再传给 js-interpreter
- 重写 `VisualizerAdapter.js`（v2）：
  - 接受 `currentSnap` + `previousSnap` 双参数
  - stack/queue 通过 diff 判断（不依赖变量名）：
    - push/pop：尾部变化且前缀匹配 → stack
    - shift：头部元素变化且其余匹配 → queue
  - linkedlist 检测：`{ val/value, next }` 结构
  - tree 检测：`{ val/value, left, right }` 结构
  - 新增 `listToFlow()` 和 `treeToFlow()` 导出函数供 VisualizerView 使用
- 重写 `VisualizerView.jsx`（v2）：
  - linkedlist / tree → React Flow 画布渲染（节点+带箭头连线）
  - array / stack / queue / object / primitive → CSS block 渲染（保留原有样式）
  - SectionLabel 新增 meta.op 标签显示（push/pop/shift/enqueue）
- 修改 `Visualizer.jsx`：读取 `prevSnap`，传递 `prevVariables` 给 VisualizerView
